# URL Shortener

A simple URL shortener web application built with Python, Flask, and HTML.

## Features
- Enter a long URL
- Generate a short URL
- Redirect to the original URL using the short link
- Simple and beginner-friendly interface

## Technologies Used
- Python
- Flask
- HTML

## Project Structure
URL_Shortener/
├── main.py
├── templates/
│   └── index.html
